Kolesa.kz

Client: Kolesa Group

Kolesa is a Custom Font and Logotype branding project

Art Director: Natalya Tritenko
Head of Design: Karina Kassimanova
CEO: Dmitry Botanov

Type Designer: Alexei Vanyashin

EN Kolesa Group manages classified ads products kolesa.kz, krisha.kz, market.kz, avtoelon.uz. In 2022 I was approached to design new logos for all products. The logo refresh builds up on existing legacy: the trademark slope in e/ is retained, and so are grotesque features.  A switch to lowercase was made to improve legibility. The logos are variable in optical and weight axes. A more neutral display corporate font was created for sub brands, and an expressive typeface for corporate merch.

RU Компания Kolesa Group управляет сервисами по размещению объявлений kolesa.kz, krisha.kz, market.kz, avtoelon.uz. В 2022 году мне предложили сделать новые логотипы для всех продуктов.  Логотип представляет собой реставрацию прежней версии: сохранена характерная /e с наклоном, сохранен стиль гротеска. Для улучшения читаемости перешли на строчные знаки вместо заглавных. Логотип вариативен: есть оптические версии для малых размеров, ось насыщенности. Для суб брендов разработано нейтральное начертание, для мерча брозкое и акцидентное.
